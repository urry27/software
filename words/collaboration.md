**Сотрудничество** <br>
**Collaboration**


Бизнес-сотрудничество - совокупность нескольких элементов внутренней активной структуры бизнеса, которые работают вместе для выполнения коллективного поведения.

Сотрудничество приложений - совокупность нескольких элементов внутренней активной структуры приложения, которые работают вместе для выполнения коллективного поведения приложения.

Сотрудничество технологий - совокупность нескольких элементов внутренней активной структуры технологии, которые работают вместе для выполнения коллективного технологического поведения.
