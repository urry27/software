**Элемент** <br>
**Element**

Активный элемент структуры:
- Внутренний активный структурный элемент представляет сущность, которая способна выполнять поведение.
- Внешний активный элемент структуры, называемый интерфейсом, представляет собой точку доступа, через которую к среде предоставляется один или несколько сервисов.

Пасивный элемент структуры:
- Пассивный элемент структуры представляет собой элемент, над которым выполняется поведение.

Элемент поведения:
- Элемент внутреннего поведения представляет собой единицу действия, которая может выполняться одним или несколькими активными структурными элементами.
- Внешний элемент поведения, называемый службой, представляет явно заданное открытое поведение.
